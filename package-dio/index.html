HTTP Request from inside WASM in Dart using package:dio

This example uses packages/dio.

See the output in browser developer console.

Actual code:


import 'package:dio/dio.dart';

final dio = Dio();
Response response = await dio.get('https://httpbin.org/anything');
print(response);

Info

  • Demo source code
  • Versions used on the moment of build: Dart SDK: v3.5.2, package:dio: v5.7.0.
