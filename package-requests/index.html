HTTP Request from inside WASM in Dart using package:requests

This example uses packages/requests, but its new version 5.0.0 from Github repo.

See the output in browser developer console.

Actual code:


import 'package:requests/requests.dart';

void main() async {
  final resp = await Requests.get('https://httpbin.org/anything');
  print('body: ${resp.content()}');
}

Info

  • Demo source code
  • Versions used on the moment of build: Dart SDK: v3.5.2, package:requests: v5.0.0.
